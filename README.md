Calculadora de IMC em Python

Este projeto é uma calculadora de IMC (Índice de Massa Corporal) desenvolvida em Python.
Ela permite ao usuário inserir seu peso e altura, mesmo usando vírgulas ou pontos, e retorna:

O valor do IMC arredondado para duas casas decimais

A classificação de acordo com as faixas do IMC:

Abaixo do peso

Peso ideal

Sobrepeso

Obesidade

Funcionalidades

Tratamento de entrada para aceitar vírgula ou ponto

Mensagens amigáveis para o usuário

Fácil de entender e modifica.
